# Bird Arcade Game 

A simple 2D arcade-style browser game inspired by classic obstacle-avoidance games, built using **HTML5 Canvas, JavaScript, and CSS**. The player controls a bird that must jump through gaps between pipes while avoiding collisions.

##  Features

* Interactive **bird jump mechanics** using keyboard or mouse input
* **Gravity-based physics** for realistic movement
* Random **pipe obstacle generation**
* **Collision detection** system
* **Score tracking** based on pipes successfully passed
* **Start screen and restart functionality**
* Responsive gameplay using the **HTML5 Canvas API**

##  Technologies Used

* HTML5
* CSS3
* JavaScript
* HTML5 Canvas API

##  How to Play

1. Open the `index.html` file in a web browser.
2. Press **SPACE** or **click** on the screen to make the bird jump.
3. Navigate the bird through the gaps between pipes.
4. Avoid hitting pipes or the ground.
5. Try to achieve the highest score possible.
